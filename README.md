# 🌤️ WTWR (What to Wear)

## 📌 Description

WTWR (What to Wear) is a React application that helps users decide what to wear based on the current weather. The app displays real-time weather data and suggests clothing items that match the temperature conditions. Users can also add their own clothing items and categorize them by weather type.

---

## ⚙️ Functionality

- Displays current weather based on location
- Recommends clothing items based on temperature
- Add new clothing items through a modal form
- Preview clothing items in a modal view
- Filter clothing by weather type (Hot / Warm / Cold)
- Dynamic rendering using React state
- Fetches weather data from an external API

---

## 🛠️ Technologies Used

- React (useState, useEffect)
- JavaScript (ES6+)
- HTML5
- CSS (BEM methodology)
- Vite
- Fetch API
- Git & GitHub
